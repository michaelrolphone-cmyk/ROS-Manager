## Springboard (Home)
### What it does
Shows the active project's name, client contact info, address, and TRS. Tiles open each mini app.

### Why it helps
Keeps the whole crew on the same page about the job location and scope before entering data.

### How to use
1. Click **Edit Details** to fill in client and location info.
2. Tap **Call**, **Email**, or **Map** to contact the client or pull up navigation.
3. Use the large tiles to jump into Traverse Builder, Points, Leveling, Evidence, Equipment, Navigation, Settings, or Help.

## Traverse Builder
### What it does
Stores traverse calls with starting coordinates, bearings, and distances, then draws a preview and produces Carlson-ready
commands.

### Why it helps
Keeps the legal description, sketch, and export in one spot so you can stake or draft without retyping.

### How to use
1. Enter a record name and click **+ New Record**.
2. Fill in starting point, coordinates, elevation, backsight azimuth, basis of bearing, and first distance.
3. Add each bearing and distance with **+ Add Subsequent Call** and reorder if needed.
4. Review the **Traverse Preview**.
5. Click **Generate Carlson Commands** to copy blocks into CAD or your collector.

## Point File Manager
### What it does
Manages point files per project, including imports, quick manual edits, traverse-to-point conversion, and downloads.

### Why it helps
Avoids juggling loose CSV/TXT files and keeps the same point numbers synced with traverses and navigation.

### How to use
1. Click **Import Points CSV** (columns: point number, X/lat, Y/lon, elevation, code, description) or start fresh with **New
   Point File**.
2. Add quick rows with **Add Point Row**.
3. Convert a traverse to points with **Generate Point File**.
4. Click **Download Points TXT** for your collector.
5. Switch or rename files using **Active Point File** and **Rename Point File**.

## Field Level Book
### What it does
Tracks backsight/foresight shots, running sums, and misclosure for a level loop with exportable notes.

### Why it helps
Replaces handwritten books, reduces arithmetic mistakes, and provides a clean PDF for the office.

### How to use
1. Create or pick a **Level run** and name it (e.g., "North ditch loop").
2. Set the starting point/elevation.
3. Add BS/FS shots in order.
4. Watch the totals and closure update.
5. Adjust until the closure matches expectations.
6. Click **Export as PDF** for your field packet.

## Evidence Records
### What it does
Attaches monument evidence to traverse points with condition, ties, photos, and optional GPS stamps.

### Why it helps
Keeps corner evidence tied to the geometry so later filings and stakeouts reference the same location data.

### How to use
1. Pick a record and traverse point.
2. Choose the evidence type and condition.
3. Describe the monument in plain language.
4. Add tie distances/bearings.
5. Upload photos.
6. Optionally use **Get GPS Location** for coordinates.
7. Save to add the entry to the project-wide list.
8. Export when you need corner filing text or backups.

## Equipment Setup Log
### What it does
Logs setup/teardown times, base heights, reference points, crews, equipment used, and optional GPS position.

### Why it helps
Provides a daybook of control setups so you can repeat or audit work without digging through separate notes.

### How to use
1. Enter setup/teardown date/time, base station height, reference point, crew member, and equipment list.
2. Capture GPS with **Get GPS Location** if available.
3. Describe the day's goals in **Work / Daily Goal**.
4. Save to list it under **Equipment Records**.

## Navigation
### What it does
Provides a live compass, target bearings, left/right offsets, and optional map preview for stakeout targets.

### Why it helps
Lets you walk to traverse points, equipment bases, or bookmarked targets without rekeying directions.

### How to use
1. Click **Start Compass** and rotate until the heading matches your target.
2. Choose a **Navigate to** target (point file entry, traverse point, or equipment source) to see distance, bearing, and offset.
3. If GPS is available, apply localization to show your location and the target on the map.
4. Use the refresh/clear buttons to update sources.

## Global Settings
### What it does
Stores reusable **Equipment Names**, **Team Members**, and **Point Codes** plus a full-app import/export.

### Why it helps
Keeps dropdown choices consistent across every project and provides a single backup file for the entire app.

### How to use
1. Add or remove values in each list so they appear in other tools.
2. Use **Download All App Data** for a full backup.
3. Use **Import App Data** to restore.
